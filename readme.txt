=== Single Page Pagination ===
Tags: pagination
Requires at least: 4.0
Tested up to: 5.4
License: GPLv2 or later

A simple plugin that displays next and previous links on single post pages. 
Posts are ordered chronologically.

== Installation ==

1. Upload `single-page-pagination` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure plugin.
4. Done!

== Updating ==

Please re-save your settings after update.
